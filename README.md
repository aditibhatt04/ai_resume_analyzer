## âœ¨ <a name="about">About</a>

**AI Resume Analyzer** is a powerful web application that provides instant, AI-powered resume analysis and feedback. Upload your resume, enter the job description, and get:

- **ATS Score** - See how well your resume performs in Applicant Tracking Systems
- **Detailed Feedback** - Analysis across 5 dimensions: Tone & Style, Content, Structure, Skills, and ATS compatibility
- **Actionable Suggestions** - Specific, practical tips to improve your resume
- **No Login Required** - Try it instantly with Quick Analysis mode
- **Save & Track** - Login to save your analyses and track improvements over time

Perfect for job seekers preparing for applications and professionals looking to optimize their resumes.

## ğŸš€ <a name="getting-started">Getting Started</a>

### Prerequisites

Ensure you have installed:
- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/en) (v18+)
- [npm](https://www.npmjs.com/)

### Installation

```bash
# Clone the repository
git clone https://github.com/aditibhatt04/ai_resume_analyzer.git
cd ai-resume-analyzer

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
# Build the project
npm run build

# Start production server
npm run start
```
